# נתונים (מקור אמת)

כאן נשמרים כל שמות ההקבצות והמורים בפורמט מובנה, כדי שנוכל:
- לחפש בקלות
- להפיק דוחות
- לייצר תיקיות ו־README אוטומטית

קבצים:
- `הקבצות.json` – רשימת כל ההקבצות לפי שכבה/מקצוע + המורה/ים + נתיב תיקייה.
- `כיתות_אם.json` – רשימת כיתות האם לפי שכבה + מחנכים.
- `תלמידים.csv` – קובץ תלמידים (כיתת אם + הקבצה לכל מקצוע).

עדכון נכון:
- משנים רק את ה־JSON
- מריצים: `python כלים/generate_structure.py`
