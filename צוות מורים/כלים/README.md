# כלים

הכלים כאן נועדו לשמור את הנתונים בצורה חכמה ועקבית: קובץ נתונים אחד, וממנו נוצרים/מתעדכנים תיקיות ו־README.

## יצירת/עדכון תיקיות ההקבצות

מקור אמת:
- `צוות מורים/נתונים/הקבצות.json`

הרצה (מתוך תיקיית `צוות מורים`):
- `python כלים/generate_structure.py`

התוצאה:
- עדכון/יצירה של `הקבצות/**/README.md`
- יצירת אינדקס: `הקבצות/INDEX.md`

## יצירת/עדכון כיתות אם

מקור אמת:
- `צוות מורים/נתונים/כיתות_אם.json`

יצירה/עדכון:
- `python כלים/generate_homerooms.py`

## דוחות

יצירה:
- `python כלים/generate_reports.py`

## תלמידים ידניים / שילובים

מקור אמת:
- `נתונים/תלמידים.csv`

יצירה/עדכון:
- `python כלים/generate_manual_group_students.py`

תוצרים:
- `הקבצות/**/תלמידים_ידני.csv`
- `דוחות/שילובים_וחריגים.md`

## סיכום כללים (חשוב)

יצירה/עדכון:
- `python כלים/generate_summary.py`

התוצר:
- `מידע_חשוב.md`

## הרצה מלאה (מומלץ)

ב-Windows PowerShell מתוך `צוות מורים`:
- `./כלים/run_all.ps1`
