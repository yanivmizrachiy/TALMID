# אתר TALMID – ניהול תלמידים לפי כיתות/הקבצות

זה אתר מקומי (Local) שעובד **באמת** עם מסד נתונים SQLite, ומסתנכרן מהנתונים שכבר קיימים אצלך בתיקיית `צוות מורים`.

## התקנה

מתוך התיקייה `צוות מורים/אתר`:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

או דרך VS Code Task: `TALMID: Setup (venv + deps)`

## סנכרון נתונים למסד נתונים

```powershell
python -m app.sync
```

או דרך VS Code Task: `TALMID: Sync DB`

המסד נוצר כאן: `צוות מורים/אתר/data/talmid.db` (מוגדר להתעלמות ב-git).

## הרצת האתר

```powershell
uvicorn app.main:app --reload --port 8000
```

או דרך VS Code Task: `TALMID: Run Web (reload)`

ואז לפתוח בדפדפן: http://127.0.0.1:8000

## מה נוצר

- דף בית: סיכום כלל בית הספר + כפתורי שכבות ז/ח/ט עם ספירת תלמידים.
- דשבורד: גרפים (Chart.js) + עומסים/גדלים.
- דף שכבה: כפתורי הקבצות עם מורה/כמות תלמידים.
- דף הקבצה: טבלה ממוספרת בסגנון אקסל (כולל כיתת אם) וקישור לעמוד תלמיד.
- דף תלמיד: מידע מלא + טופס לשמירת ציונים/מדדים (נשמר ב-SQLite).
- דף מורה: כל ההקבצות של המורה וכמות תלמידים בכל אחת.
- דף כיתת אם: מחנכים + טבלת תלמידים וההקבצה במתמטיקה.
- דף מידע חשוב: מציג את `צוות מורים/מידע_חשוב.md` בצורה קריאה.
